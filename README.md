# 데모 영상 또는 배포 링크

데모 영상은 배포 링크로 대체 가능하며, 배포가 되었고 배포된 사이트에서 기능이 모두 동작하면 가산점이 부여됩니다.
배포된 사이트에서 기능이 정상동작 하지 않는다면 배포 가산점이 부여되지 않습니다

기능이 정상 동작하지 않는 예시:
새로고침하면 404 에러 페이지 표출
"/" URL이 아닌 "/signup"등의 경로로 바로 접속할 경우 404 에러 페이지 표출 등
